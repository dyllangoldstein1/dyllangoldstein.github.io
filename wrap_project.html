Wireless RF & Analog Communication System

An end-to-end project to design, build, and test a 27MHz wireless communication system from the ground up.

Project Overview

The Wireless, RF, and Analog Project (WRAP) is an electrifying undertaking to build a complete digital communication system. The main goal is to successfully transmit a string of bits from one microcontroller to another over a 27MHz radio frequency link.

This project covers the entire signal chain, from writing the embedded code that encodes digital data into an analog waveform, to designing and building the physical transmitter and receiver hardware, and finally, decoding the analog signal back into an error-free digital format.
