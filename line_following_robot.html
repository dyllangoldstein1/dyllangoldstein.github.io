Line-Following Robot

An autonomous robot programmed to navigate a course by following a black line on a white surface. *Please note the video below was not my personal robot but rather a classmates as I hadn't taken a video of my personal robot completing the project.*

Project Overview

This project had the goal of programming a vehicle that could autonomously navigate a random path with pre-defined challenges. The robot used IR sensors located on its underside to detect a black-line and decision making algorithms programmed in to address challenges such as split or lost paths.

Having to ensure the robot reacted to obstacles with varying demands posed a great challenge with some requiring sharp turns while others required avoiding an overcorrection. This was overcome with a weighting scheme and path recovery algorithm. The weighting scheme ensured that when encountering two potential paths, the robot would prioritize the path to its left side. This implementation was reveresed when returning. Additionally, a path recovery algorithm was crucial to accomplishing "jumps" where the line stopped and moved to the side. This way when the robot would lose sight of the track with would slowly drift until it once again found the line. From there PID controls took over and ensured the robot stayed on course.

Core Concepts & Implementation

The robot's functionality is based on three key subsystems:

  • Sensor Array: An array of infrared (IR) sensors used to detect the line. The white surface reflects IR light, while the black line absorbs it. Using analog values from the phototransistors, robot could determine its position relative to the line.
  • Motor Control: A motor driver allows the microcontroller to precisely control the speed and direction of the two DC motors connected to the wheels. By varying the Pulse Width Modulation (PWM) signal sent to each motor, the robot was able to accomplish sharp turns, gradual curves, or move straight.
  • Control Logic: The core of the project was the implementation of conditional logic, weighting schemes, and Proportional-Integral-Derivative (PID) controls to create a smooth and fast navigation.

Anticipated Outcome & Skills Gained

The final outcome was a fully autonomous robot capable of navigating a complex track with curves and intersections. The project provided hands-on experience in microcontroller programming and control system theory.
