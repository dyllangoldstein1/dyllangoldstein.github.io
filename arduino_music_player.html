Arduino 'iPoduino' Music Player

A hands-on project building a music player with programmable songs, a potentiometer interface, and a custom LED display.

Project Overview

This project involved building a standalone music player, nicknamed the "iPoduino," on a breadboard using an Arduino Nano. The device integrates three distinct subsystems: an audio output system that plays programmed melodies through a speaker, a user interface using a potentiometer to select one of three songs, and a custom-designed visual display using LEDs to provide feedback on the current song selection.

The goal was to gain practical experience with core electrical engineering concepts by managing digital and analog signals to create a functional, interactive device.

Core Concepts & Implementation

The central concept of this project is **Pulse Width Modulation (PWM)**. Since the Arduino cannot produce a true analog signal, PWM is used to simulate one. This technique was fundamental to multiple subsystems:

  • Audio Generation: The `tone()` function was used to generate square waves of specific frequencies, which the speaker translates into audible musical notes. The duration of each note was controlled by programming delays between `tone()` and `noTone()` calls.
  • Visual Display: PWM was also used to control the brightness of the LEDs in the display, allowing for more dynamic feedback than simply turning them on or off.

A significant challenge was the **system integration** of the different hardware components. The final program reads an analog voltage from the potentiometer's wiper, maps that voltage value to a specific song, updates the LED display to reflect the choice, and then plays the correct sequence of notes through the speaker upon a reset-button press. This required careful code organization, including creating separate functions for each song.

Final Outcome & Skills Demonstrated

The result is a fully functional, breadboard-based music player. The user can turn the potentiometer dial to select a song, see their selection confirmed on the LED display, and press the Arduino's reset button to play the chosen melody.

This project successfully demonstrates a practical understanding of **Arduino programming (C/C++)**, analog and digital I/O, implementing **PWM** for audio and visual outputs, reading sensor data via **Analog-to-Digital Conversion (ADC)**, and building and debugging a multi-component circuit.
